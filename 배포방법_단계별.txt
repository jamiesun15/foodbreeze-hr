================================================================================
         foodbreeze.vercel.app 무료 도메인 연결 방법
         (완전 초보자용 - 따라하기만 하면 됩니다!)
================================================================================

총 3단계입니다. 약 20~30분 소요됩니다.

================================================================================
STEP 1 : GitHub 계정 만들고 코드 올리기
         (GitHub = 코드를 인터넷에 저장하는 곳. 무료)
================================================================================

1-1. 크롬에서 github.com 접속 → [Sign up] 클릭
     이메일, 비밀번호 입력 후 계정 만들기

1-2. 로그인 후 오른쪽 위 [+] → [New repository] 클릭
     Repository name: foodbreeze-hr
     Public 선택 → [Create repository] 클릭

1-3. 터미널(PowerShell)을 열고 아래 명령어를 순서대로 입력:

     cd C:\Users\USER\Desktop\HRtest\hrapp

     git init
     git add .
     git commit -m "푸드브리즈 HR 시스템 첫 배포"
     git branch -M main
     git remote add origin https://github.com/여기에_본인_깃허브아이디/foodbreeze-hr.git
     git push -u origin main

     ※ 여기에_본인_깃허브아이디 → 본인이 만든 GitHub 아이디로 교체

================================================================================
STEP 2 : Neon 데이터베이스 만들기
         (Neon = 무료 클라우드 데이터베이스)
================================================================================

2-1. 크롬에서 neon.tech 접속 → [Sign up] 클릭 (GitHub 계정으로 가입 가능)

2-2. 로그인 후 [Create Project] 클릭
     Project name: foodbreeze-hr
     Database name: foodbreeze
     Region: AWS Asia Pacific (Singapore) 선택  ← 한국과 가장 가까움
     [Create Project] 클릭

2-3. 프로젝트가 만들어지면 화면에 연결 주소가 나옵니다.
     [Connection string] 탭 클릭
     [.env] 선택하면 이런 형태의 주소가 나옵니다:

     DATABASE_URL=postgresql://user:password@ep-xxx.ap-southeast-1.aws.neon.tech/foodbreeze?sslmode=require

     이 전체 줄을 복사해 두세요! (나중에 필요합니다)

================================================================================
STEP 3 : Vercel에 배포하기
         (Vercel = 무료 웹 호스팅. foodbreeze.vercel.app 주소 받는 곳)
================================================================================

3-1. 크롬에서 vercel.com 접속 → [Sign Up] → [Continue with GitHub] 클릭
     GitHub 계정으로 로그인

3-2. [Add New...] → [Project] 클릭
     GitHub 저장소 목록에서 [foodbreeze-hr] 찾아서 [Import] 클릭

3-3. Project Name 에 입력: foodbreeze
     (이게 나중에 foodbreeze.vercel.app 주소가 됩니다!)

3-4. [Environment Variables] 섹션을 열고:
     NAME  : DATABASE_URL
     VALUE : (STEP 2에서 복사한 postgresql://... 주소를 붙여넣기)
     [Add] 클릭

     NAME  : SESSION_SECRET
     VALUE : foodbreeze-hr-super-secret-key-2024
     [Add] 클릭

3-5. [Deploy] 버튼 클릭!
     2~3분 기다리면 완료됩니다.

================================================================================
STEP 4 : 데이터베이스 초기화 (딱 1번만!)
================================================================================

배포 완료 후 크롬에서 아래 주소 접속:
https://foodbreeze.vercel.app/api/setup

화면에 이런 메시지가 나오면 성공:
{"success":true,"message":"데이터베이스 설정 완료!...","admin":"admin / admin1234"}
