================================================================================
             푸드브리즈 HR 시스템 - 시작 방법
================================================================================

★ 처음 시작하는 방법 ★

1. 이 폴더(hrapp)에서 마우스 오른쪽 클릭
   → "터미널에서 열기" 또는 "PowerShell에서 열기" 클릭

2. 아래 명령어 입력 후 엔터:
   npm run dev

3. 인터넷 브라우저(크롬 권장)에서 주소창에 입력:
   http://localhost:3000

4. 로그인 화면이 나오면:
   아이디: admin
   비밀번호: admin1234

★ 처음 할 일 ★
1. 로그인 후 [직원 관리] → [직원 등록]에서 직원을 추가하세요
2. 각 직원의 기본급을 입력해야 급여 계산이 됩니다
3. 매월 급여일 전에 [급여 관리] → [일괄 급여 계산] 버튼을 누르세요

================================================================================
★ 기능 안내 ★

1. 대시보드     : 오늘 출근 현황, 직원 수, 급여 합계 한눈에 보기
2. 직원 관리    : 직원 추가/수정/퇴사 처리
3. 출퇴근 관리  : 출근/퇴근 버튼 클릭 (직원 본인도 가능), 기록 수정
4. 급여 관리    : 매월 급여 자동계산 (4대보험+소득세 자동), 명세서 출력
5. 세금계산서   : 거래처 세금계산서/계산서/영수증 발행
6. 보고서       : 월별 출퇴근 현황, 직원별 급여 통계

================================================================================
★ 비밀번호 변경 방법 (나중에) ★
관리자 페이지에서 변경 예정 (추가 개발 예정)

================================================================================
★ 스마트폰에서 접속하는 방법 ★
1. 컴퓨터와 같은 와이파이에 연결되어 있어야 합니다
2. 컴퓨터의 IP 주소 확인 (cmd에서 ipconfig 입력)
3. 스마트폰 브라우저에서: http://[컴퓨터IP]:3000
